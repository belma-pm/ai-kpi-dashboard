# AI-Powered KPI Dashboard

This project is an interactive Streamlit dashboard designed to analyze
business performance using key metrics such as Revenue, Orders,
Average Order Value (AOV), risk scoring, and trend-based forecasting.

## Features
- CSV upload & validation
- Month-over-Month KPI analysis
- Risk scoring engine
- Revenue trend & forecast visualization
- AI-style executive commentary (simulated)

## Tech Stack
- Python
- Pandas
- NumPy
- Streamlit
- Plotly

## Expected CSV Format
Month | Revenue | Orders
